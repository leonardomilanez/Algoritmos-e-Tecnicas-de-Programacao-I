# Algoritmos-e-Tecnicas-de-Programacao-I
Esta matéria aborda os conceitos fundamentais de lógica de programação, estruturas de dados lineares e alocação de memória. 

O objetivo principal destes arquivos é registrar o meu aprendizado prático e a evolução na linguagem C++.

## Tecnologias e Ferramentas Utilizadas

* **Linguagem Principal:** C++
* **Compilador/Runtime:** GCC.
* **IDE/Editor:** VS Code
  
## Tópicos Estudados

* Lógica de Programação
* Estruturas de dados lineares (arrays e matrizes)
* Registros (structs)
* Ponteiros
